Abstract

We test whether a simple threshold gating (IG: inhibitory gating implemented as a lower firing threshold) increases collective responsiveness in a small spiking neural network. In a 5‑neuron LIF network receiving the same pulse input (0.1–0.7 s), total spikes were 0 for baseline (α = 1.0) and 86 for IG‑on (α = 0.7), indicating a qualitative transition from no activity to broad network firing under identical input.

ConditionTotal spikes (1 s)

Interpretation

A 30% reduction in threshold drives a regime shift from “no firing” to “collective firing”. We interpret this as (i) single‑neuron threshold approach combined with (ii) recurrent excitatory positive feedback, yielding a collective threshold phenomenon.

Threshold Gating in a 5‑Neuron LIF SNN — Summary & Next Steps (p.2)

Author: GNJz · jazzin@qig.ai · 2025‑09‑02
Code: QIG/code/lif_network.py (commit fc6ceee)

Five‑line Summary

  • With identical input (0.1–0.7 s), lowering threshold (α = 0.7) yields a regime shift to collective spiking.
  • Total spikes: baseline 0 vs. IG‑on 86 (N=5, T=1 s, dt=1 ms).
  • Mechanism: (i) approach to threshold + (ii) recurrent positive feedback.
  • Minimal gating yields large changes in sensitivity/synchrony → potential task gains.
  • Current result is a minimal reproduction; statistical validation and sweeps required.

Reproduction (local)

cd ~/Desktop/Qquarts_Lab/QIG/code
python3 lif_network.py   # generates figs/raster_baseline.png and figs/raster_ig.png

Risks & Checklist

  • Single seed, small N=5 → parameter/seed sensitivity (need multiple seeds).
  • Baseline = 0 spikes → avoid ratios; use absolute differences, response/no‑response, latency, firing‑neuron ratio.
  • Simple input/connectivity → task generalization to be checked separately.

Next Steps (toward a paper)

  • Reproducibility: 30 seeds, 95% CI.
  • Sweeps: α ∈ {0.5…1.0}, N ∈ {5, 20, 50}, input strength/length.
  • Metrics: total spikes, latency, firing‑neuron ratio, synchrony/Fano.
  • Ablation: cut recurrence (W=0), manipulate inhibition, modify input window.
  • Task: simple binary detection; compare IG on/off performance.
