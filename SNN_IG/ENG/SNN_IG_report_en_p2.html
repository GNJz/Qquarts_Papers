Key findings (5 lines)

  • With identical input (0.1–0.7 s), threshold gating (α=0.7) alone triggers a no‑spike → network‑spike transition.
  • Total spikes: baseline 0 vs. IG on 86 (N=5, T=1 s, dt=1 ms).
  • Transition plausibly arises from (i) near‑threshold membrane dynamics + (ii) positive feedback in recurrence.
  • A simple gating control markedly shifts sensitivity/synchrony, suggesting task‑level benefits.
  • This is a minimal reproducible example; robust statistics and tasks are required for claims.

Risks & checklist

  • Single seed, N=5 → sensitivity to parameters/randomness (repeat with multiple seeds).
  • Baseline=0 complicates ratio‑based effect sizes → track response/no‑response, latency, fraction of spiking units.
  • Simplified input/connectivity → task relevance requires an explicit downstream task.

Planned experiments

  • Statistical replication: 30 seeds, report mean ± 95% CI.
  • Sweeps: α ∈ {0.5…1.0}, N ∈ {5,20,50}, input strength/length.
  • Metrics: total spikes, response latency, spiking fraction, synchrony/Fano.
  • Ablations: remove recurrence (W=0), remove/boost inhibition.
  • Task suitability: simple binary detection; compare IG on/off.
