핵심 요약 (5줄)

  • 동일 입력(0.1–0.7 s)에서 임계값 게이팅(α=0.7)만으로 무발화 → 집단 발화 전이 확인.
  • 총 스파이크: baseline 0 ↔ IG on 86 (N=5, T=1 s, dt=1 ms).
  • 전이는 (i) 임계 접근 + (ii) 재귀 흥분성 양의 피드백 결합의 결과로 해석.
  • 간단한 게이트만으로 네트워크 민감도/동조 크게 변화 → 태스크 성능 잠재력.
  • 현재는 최소 재현 예시; 통계·스윕·태스크 검증 필요.

다음 단계 (실험 계획)

  • 통계 재현: 시드 30개, 95% CI.
  • 스윕: α ∈ {0.5…1.0}, N ∈ {5,20,50}, 입력 강도/길이.
  • 지표: 총 스파이크, 반응 지연, 발화 뉴런 비율, 동조도/Fano.
  • Ablation: 재귀 차단(W=0), 억제 링크 제거/증가.
  • 태스크: 간단한 이진 감지에서 IG on/off 성능 비교.
